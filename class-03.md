# LISTS

* Ordered lists    
> use `<ol><li></li></ol>`  

* Unordered lists  
> use `<ul><li></li></ul>`  

* Definition lists  
> use `<dl><dt></dt></dl>` or `<dl><dd></dt></dd>`  

* Nested list  
> use alist inside another list
